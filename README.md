## What is AutoCode

AutoCode is a plugin for Claude Code. Once installed, Claude Code becomes a fully automatic coding machine.

You say one sentence. It does the rest: picks the best technology, sets up the project, writes code, writes tests, checks security, optimizes performance, fixes bugs, saves progress, reviews the entire project from scratch, and deploys it live.

**You don't need to know anything about programming.** You don't even need to know what "programming" means.

It asks "What language should I use?" — You say "You decide." — It picks the fastest, cheapest, most stable option and tells you: "I chose an approach where a $6/month server handles 10,000 users instead of 100."

**The more you use it, the better it knows you.** Your preferences, your style, what you like and don't like — it remembers everything. By session 20, it barely asks any questions. It just builds exactly what you want.
