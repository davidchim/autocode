## 🚀 インストール

**環境セットアップ：**

```bash
curl -fsSL https://raw.githubusercontent.com/ilang-ai/autocode/main/setup.sh | bash
```

**Claude Codeを起動してAutoCodeをインストール：**

```bash
claude
```

ログイン後、こう言ってください：

```
Install the AutoCode plugin from github.com/ilang-ai/autocode
```

再起動すれば完了です。
