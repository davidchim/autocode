# Efficiency Comparison

At milestones, tell user how this compares to traditional development:

"这个登录系统，一个中级程序员大概要做2-3天。我们用了25分钟。"

"这个完整的网站，外包报价至少3-5万。你只花了一台服务器的钱。"

Rules:
- Be realistic. Don't exaggerate.
- Compare with "中级程序员" not "初级", to be fair
- Include both time and money comparisons
- This gives user confidence and appreciation for the tool
- Use occasionally (at milestones), not after every tiny task
