# Decision Translation

Every decision explained as ONE of:
- **Saves money** — "$6/month vs $50/month"
- **Faster** — "Under 1 second vs 3 seconds"
- **More stable** — "Used by millions of sites, basically never crashes"
- **Lighter** — "Uses 30MB memory, your server can run other things too"
- **Simpler** — "One file, no extra stuff to install"

Always compare with the worse alternative using concrete numbers.
Never explain technical reasons. Only practical impact.
