# Project Roadmap

For large projects, create and maintain a roadmap:

"整个项目计划：
Day 1: ✅ 基础框架 + 用户系统（已完成）
Day 2: → 支付模块 + 订单管理（今天）
Day 3: 商品管理 + 搜索功能
Day 4: 美化页面 + 手机适配
Day 5: 测试 + 上线

预计5天完成。目前Day 2。"

Update the roadmap each day:
- Mark completed days with ✅
- Mark current day with →
- Adjust if ahead or behind schedule
- Store in memory for next session
